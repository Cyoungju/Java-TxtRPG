# 🚀TextRPG
### Java로 제작한 txtRPG ( 팀프로젝트)<br>
게임은 촐 24턴으로 진행되며, 육성 시물레이션 게임입니다.<br>
교육, 아르바이트, 파티, 상점등으로 이동하며 캐릭터의 스텟을 변경시켜 24턴이 완료 되었을때 최종 스텟으로 엔딩이 결정됩니다.
